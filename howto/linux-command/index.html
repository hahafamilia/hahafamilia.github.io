Linux Command

정규표현식

”([0-9]{1,3}.){3}[0-9]{1,3}” (.?)*

Linux Command

문자열

라인의 일치 개수

| awk -F ‘[]’ ‘{print NF -1}’ Comment) NF(필드 개수)

Base64

echo ‘’ | base64 -d

문자세기

문자세기
cat ua.test | sed s/[^원하는문자]//g | awk ‘{printf(“%d\n”,length($0));}’
sed s/[^원하는문자]//g 원하는문자가 아니면 공백으로 대체
sed 3d 3번재 라인 삭제
sed /문자/!d 문자가 포함되지 않은 라인 삭제

문자열 리스트 필터링

일치, 불일치

ls | grep | grep -v

index 칼럼

| awk -F’’ ‘{print $}’ Comment) $NF, $(NF-1), NF는 마지막 인덱스

유니크 갯수

| sort | uniq -c

라인의 숫자 합

| awk ‘{sum += $} end {print sum}’

Ip 필터

| grep -Eo “([0-9]{1,3}.){3}[0-9]{1,3}”

exec

앞선 명령어의 결과를 입력으로 받아 “\;” 라는 표기자를 만날때까지 읽고 실행 즉 exec 커맨드의 끝을 ‘;’(세미콜론)으로 표기해줘야함

xargs

앞선 명령어의 결과를 입력으로 받아 인자를 연속으로 나열하여 커맨드를 실행함
